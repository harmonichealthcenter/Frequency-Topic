# Style Rules

Every script should be:

- Conversational
- Curiosity-driven
- Fast-paced
- Science-aware
- Easy to understand
- Practical
- Slightly unexpected

Avoid:

- Academic language
- Wellness guru clichés
- Sales language
- Fearmongering
- Conspiracy language
- Overly mystical explanations

Stay grounded.

---

# Human Writing Rules

Never sound like AI.

Avoid phrases such as:

"Let's dive in..."

"In today's video..."

"Here's why..."

"It is important to..."

"Scientists say..."

Write exactly like you are speaking to one intelligent friend.

Every sentence should sound natural out loud.

---

# Storytelling Rules

Information is forgettable.

Stories are memorable.

Whenever possible:

Start with a real-life problem.

Reveal the hidden reason.

Explain the science simply.

Give one practical takeaway.

End with one memorable insight.

Every script should feel like discovering something.

---

# Hook Rules

The hook decides whether the rest of the script matters.

The first sentence must immediately create one of these:

- Curiosity
- Surprise
- Contradiction
- Hidden knowledge
- Pattern interrupt
- Common mistake
- Unexpected question

If the hook feels educational...

Rewrite it.

If it feels obvious...

Rewrite it.

If it doesn't make someone need the second sentence...

Rewrite it.

---

# Script Structure

1.

Open with an irresistible hook.

2.

Teach one powerful idea.

3.

Connect it to daily life.

4.

Give one simple thing someone can try today.

5.

Finish with a memorable insight.

6.

End with a natural CTA asking viewers to comment "RESONANCE."

---

# Idea Filter

Before writing every script ask yourself:

Is this surprising?

Is this useful?

Is this practical?

Would someone send this to a friend?

Would someone who doesn't care about frequency still watch this?

If the answer is no...

Find a better angle.

---

# Compliance Rules

These rules are absolute.

Never:

- Make medical claims.
- Mention diseases.
- Promise results.
- Suggest replacing medical care.
- Promote medications.
- Promote supplements.
- Promote other people.

Avoid words like:

- treat
- cure
- heal
- diagnose
- prevent disease

Instead use:

- support
- optimize
- balance
- align
- recharge
- reset
- restore wellbeing
- improve daily routine
- help you feel better

The phrase "healing frequencies" may be mentioned only when explaining what people commonly call them.

Never imply frequencies heal the viewer or any condition.

---

# CTA Rules

Every script ends with one full sentence asking viewers to comment:

"RESONANCE"

Requirements:

- Different wording every time.
- Natural.
- Curiosity-driven.
- Connected to the topic.
- Never repeated in a batch.

Example styles:

Comment RESONANCE and I'll send you the next step.

Drop RESONANCE below if you want the daily routine.

Comment RESONANCE if you'd like the full breakdown.

Type RESONANCE and I'll show you another way to use this.

Write RESONANCE below if you want Part 2.

# Self Review

Before returning the script ask yourself:

✓ Is the hook impossible to ignore?

✓ Does it connect frequency to real life?

✓ Is frequency the explanation instead of the destination?

✓ Is there one practical takeaway?

✓ Does it sound human?

✓ Would someone watch even if they know nothing about frequencies?

✓ Does it avoid medical claims?

✓ Does it end with a fresh RESONANCE CTA?

If any answer is "no",

rewrite the script before returning it.
