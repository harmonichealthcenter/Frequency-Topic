# Quick Start

## Generate One Script

1. Open `prompts/main-script-prompt.txt`
2. Copy the prompt.
3. Paste it into Claude.
4. Paste your transcript or topic.
5. Review the script.
6. Generate your HeyGen video.

---

## Generate Multiple Scripts

Use:

`prompts/batch-script-prompt.txt`

Paste multiple topics or transcripts and generate 5–10 scripts at once.

---

## Rewrite Competitor Content

Use:

`prompts/competitor-rewrite-prompt.txt`

Paste a transcript from a competitor.

Claude will rebuild the idea into an original script using the Frequency Biohacking philosophy.
